# Stock_analysis

---

##  Project Overview

| Detail | Info |
|---|---|
| **Dataset** | Simulated daily stock prices (252 trading days) |
| **Stocks** | Apple, Google, Amazon, Tesla, Microsoft |
| **Period** | January – December 2024 |
| **Tools** | Microsoft Excel |

---

## Key Questions Answered

1. Which stock delivered the best risk-adjusted return (Sharpe Ratio)?
2. How did each stock perform quarter by quarter?
3. Which months were the best and worst for each stock?
4. Is there a trade-off between volatility and total return?
5. How do the stocks compare on a normalized basis from Day 1?

---

## Key Findings

- **MSFT** had the highest Sharpe Ratio — best risk-adjusted return of the group
- **TSLA** was the most volatile stock with the widest monthly swings
- **Q4** was the strongest quarter across most stocks
- Higher volatility (TSLA, AMZN) did not always translate to higher returns
- **AAPL & MSFT** showed the most consistent month-over-month performance
