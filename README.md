# Stock Market Performance Analysis — 2024

An end-to-end analysis of 5 major stocks (AAPL, GOOGL, AMZN, TSLA, MSFT) over the full year 2024, built across two tools to demonstrate a complete data analyst workflow.

---

## Project Overview

| Detail | Info |
|---|---|
| **Dataset** | Simulated daily stock prices — 252 trading days |
| **Stocks** | Apple, Google, Amazon, Tesla, Microsoft |
| **Period** | January – December 2024 |
| **Tools** | Python (Pandas, Matplotlib) · Microsoft Excel |

---

## Key Questions Answered

1. Which stock delivered the best total return over the year?
2. Which stock had the best risk-adjusted return (Sharpe Ratio)?
3. How did performance vary quarter by quarter?
4. Which months were strongest and weakest for each stock?
5. How correlated are these stocks with each other?
6. Is there a clear trade-off between risk and return?

---

## Key Findings

- MSFT had the highest Sharpe Ratio — strongest risk-adjusted return in the group
- TSLA was the most volatile stock with the widest monthly swings
- Q4 was the strongest quarter across most stocks
- Higher volatility did not consistently translate into higher returns
- All five stocks show moderate-to-high correlation, limiting diversification within this group

## Python Analysis — How to Run

1. Clone the repo and navigate to the folder
   ```bash
   git clone https://github.com/liasaiyan/stock-market-analysis.git
   cd stock-market-analysis
   ```

2. Install dependencies
   ```bash
   pip install pandas numpy matplotlib jupyter
   ```

3. Launch the notebook
   ```bash
   jupyter notebook stock_analysis.ipynb
   ```

4. Run all cells — data is generated automatically, no external files needed
